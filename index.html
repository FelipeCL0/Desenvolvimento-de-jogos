Desenvolvimento de Jogos

Oque é um jogo?

Jogos são aplicativos interativos que buscam entreter o usuario. sendo assim, os desenvolvedores Buscam desenvolver jogos divertidos para tal publico alvo

Existem vários generos de jogos,entre eles estão:

educativos: buscam educar o usuario de uma determinada idade.

Competitivos: os jogos competitivos são jogos que os jogadores competem entre si, para ganhar alguma

recompensa in-game ou até mesmo com dinheiro real, como é o caso em campeonatos de jogos competitivos.

Cooperativos: são jogos que os jogadores precisam cooperar entre si para ganhar alguma recompensa.

Etc...

Como um jogo é feito?

Um jogo pode ser feito de varias maneiras, uma delas é pela programação.

A programação consiste em palavras que o computador interpleta e executa no programa de uma maneira mais

simples, é a lingua dos computadores

Existem varias linguas de programação, que podem servir para diferentes objetivos

uma das linguagens mais usadas no mundo da programação é o JavaScript, com ele você pode criar sites, jogos ou

programas.
