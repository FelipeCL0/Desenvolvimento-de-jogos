Desenvolvimento de Jogos

Oque é um jogo?

Jogos são aplicativos interativos que buscam entreter o usuario. sendo assim, os desenvolvedores

Buscam desenvolver jogos divertidos para tal publico alvo

Existem vários generos de jogos,entre eles estão:

educativos: buscam educar o usuario de uma determinada idade.

Competitivos: os jogos competitivos são jogos que os jogadores competem entre si, para ganhar alguma

recompensa in-game ou até mesmo com dinheiro real, como é o caso em campeonatos de jogos competitivos.

Cooperativos: são jogos que os jogadores precisam cooperar entre si para ganhar alguma recompensa.
